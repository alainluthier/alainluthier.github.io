Visual Hierarchy

Covid Visualizer

www.covidvisualizer.com

This website has a good design and applies the visual hierarchy principle because the first impression is the Earth's globe figure and it explains the purpose of the website, share information about global COVID-19 infection.

Hick’s Law

Zoom

www.zoom.us

This website uses Hick’s law principle because there are two buttons, and the user has two choices only. It is easier for the user to decide what he wants to do.

PARC: Repetition

Arduino

www.arduino.cc

This website uses the Repetition principle because the background color of the heading is the same as many buttons and icons. It unifies the visual appeal.
