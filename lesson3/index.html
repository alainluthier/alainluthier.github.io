Site Name

The site is going to be called digitalweather.org. It will be a website with digital information about weather in the world.

Site purpose

The main objective of the digitalwatherstation.com website will be to share information about the world's climate, regarding important news, cities with major meteorological events, and the real-time weather situation. This objective will be achieved through the detailed description of major meteorological events in the world and with the weather forecast for cities that will allow users to use the data for analysis and information.
